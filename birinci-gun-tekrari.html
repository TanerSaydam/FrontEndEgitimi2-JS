JavaScript 1.Gün Tekrarı

1995 yılında Netspace çalışanı tarafından bulunmuştur

Browserlarda bulunan JavaScript motoru sayasinde çalıştırırlar

1997 yılında internet explorer browserında kullanmıya başlandığı için diğer borwserlar ile aynı özelliklere sahip olsun ve yeni çıkacak javascript versiyonlarında uyumsuzluk yaşanmasın diye ECMAScript standardı getirildi. Kısaca JavaScript versiyonları ECMAScript standardı olarak yayınlanır. Bu standardaki bilgileri tüm JavaScript destekleyen browserlar uygular.

Günümüzde en son ES13 versiyonu var. Piyasada en çok konuşulan versiyonu ise ES6'dır.
